# Getting Started

## Installing

You can install react-tailwind-uikit with [NPM](https://www.npmjs.com/package/react-tailwind-uikit) or Yarn.

- NPM: `npm install --save react-tailwind-uikit`
- YARN: `yarn add react-tailwind-uikit`

## Configuration

For Basic Configuration in your project follow these steps [Installing Steps](https://tailwindcss.com/docs/installation) According to your next js or react js Project

## Compatibility

react-tailwind-uikit is compatible with React v16+ and works with ReactDOM. React Native is not supported at this time.It will support nextjs Project also.This framework used tailwind css , Typescript ,React js .
