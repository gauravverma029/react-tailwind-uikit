## Quick Start

To create a Button, use import the `Button`. The Button
will automatically configure itself with the default options exposed via `props`.

```jsx
import { Button } from "react-tailwind-uikit";

export default () => <Button>Submit</Button>;
```

## With Different variant

use different variant check api props for this

```jsx
import { Button } from "react-tailwind-uikit";

export default () => <Button variant="secondary">Submit</Button>;
```
